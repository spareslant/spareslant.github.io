Saltstack: Call runner from minion (jinja conditionals)

2020-03-15
Introduction Sometimes a minion may need to execute something which is central to whole system but based on some condition. e.g. Minion may need to update an external common Database whose write access is given only to master. Therefore minion may need to ask salt-master to execute on its behalf. This can be done by Minion calling runner on salt-master runner executing a custom execution module that provides access to DB As a custom execution module needs to be executed on salt-master, therefore salt-minion also needs to be installed on salt-master.
